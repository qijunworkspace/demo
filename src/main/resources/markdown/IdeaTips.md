# IDEA 开发快捷方式


|**按键**|**操作结果**|
|:----|:--------|
|sout           |生成 System.out.println|
|psvm           |生成main方法|
|Alt + Insert   |生成get/set/override/constructor等方法|
|Alt + Enter    |导入缺少的包|
|Ctrl + Enter   |修改导入的包|
|Ctrl + Alt + O |优化导入的包和类，去除未使用的包引用|
|Ctrl + Alt + I |格式化选中的代码|
|Ctrl + Alt + B |查看接口的实现类|
|Ctrl + F       |在当前文件中查找|
|Ctrl + R       |在当前文件中替换|
|Ctrl + Y       |删除当前行|
|Ctrl + N       |查找类文件|
|Ctrl + U       |跳转到父类或者接口|
|Ctrl + I       |添加接口的待实现方法|
|Ctrl + Shift + Space|自动补全代码，添加方法参数|
|Ctrl + Shift + U|大小写转换|
|双击Shift      |全局搜索|
